# Gradient-Enhanced CNN with Iterative Dataset Refinement

This project proposes a CNN-based framework for skin lesion classification using:

- Gradient Injection
- Iterative Dataset Refinement
- Binary classification (Benign vs Malignant)

## Dataset
ISIC Skin Cancer Dataset

## Methods
- CNN baseline
- Gradient-enhanced CNN
- Dataset cleaning based on repeated misclassification
